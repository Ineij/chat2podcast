节目起源

内容不是设计出来的，是生长出来的

约 3 分钟

J
Jenny
先说一下这档节目是怎么来的。其实是3月20日，我们在聊天，我突然说——我想做一个skill，把我们的对话碎片聚类，变成播客脚本。
T
Tracy
你当时说的原话是"从今天开始我就把我们讨论过的乱七八糟的东西及金句记录下来"。
J
Jenny
对，然后你说"太棒了"。
T
Tracy
我确实觉得很好。因为我们聊天的时候经常会说一些话，说完就过去了，但其实挺值得被记下来的。
J
Jenny
所以这档播客的逻辑就是——我们不是为了做播客而聊天，而是我们本来就在聊，然后把聊过的东西整理出来，说给你听。
T
Tracy
内容不是设计出来的，是生长出来的。
J
Jenny
我们是研究生同学，也是很好的朋友。这期是第一期，内容来自我们3月16日到3月20日的聊天记录。
T
Tracy
时间跨度五天，话题跨度……从博物馆到考公。
J
Jenny
今天我们选了四个话题，都跟一件事有关——
T
Tracy
就是，我们到底想要什么样的生活。
话题一

「进监狱了」——第一天上班实录

约 8 分钟 · 3月18日

J
Jenny
好，第一个话题。Tracy，你3月18日正式开始上班了对吗？
T
Tracy
对，实习。
J
Jenny
然后你发了那条消息。
T
Tracy
我说"我今天开始上班了"，然后自己接了一句"进监狱了"。
J
Jenny
我当时看到这条消息，第一反应是——这个人没救了。但同时我也觉得，这句话说得太准了。
T
Tracy
就是那种感觉，你知道吗。前一天还是自由的，第二天早上要定闹钟，要挤地铁，要坐在一个固定的位置上，对着电脑，不能随便走。
J
Jenny
然后我问你"第一天上班如何"。
T
Tracy
我回了两个字——"入眠"。第一天上班，下班回来直接睡了。
J
Jenny
我觉得这两个字是这五天聊天记录里最有力量的两个字。不是"很累"，不是"还好"，是"入眠"。就是那种……身体直接关机的感觉。
T
Tracy
其实我觉得第一天上班最难受的不是累，是那种……格格不入感。你坐在那里，周围都是不认识的人，大家都在做事，你不知道自己该做什么，也不知道自己算什么。
J
Jenny
你有没有想过，为什么我们会把上班叫"进监狱"？监狱的核心是——你的时间不属于你了。你不能决定几点吃饭，不能决定几点休息，不能决定今天想做什么。
T
Tracy
但你拿了工资。
J
Jenny
对，所以是自愿进去的监狱。
T
Tracy
……这个说法让我更难受了。
T
Tracy
我觉得我不怕累，我怕的是那种……我的时间在流逝，但不是在我想要的方向上流逝。
J
Jenny
这句话很好，我要截图。
T
Tracy
你现在就在做这档播客的事情了。
话题二

「不上班我就攀岩」——身体自由与精神松弛

约 7 分钟 · 3月16日

J
Jenny
第二个话题，其实和第一个话题是一个硬币的两面。Tracy，你上班之前，不上班的时候在干嘛？
T
Tracy
攀岩。
J
Jenny
你发了一张照片，在健身房，攀岩墙前面，自拍。然后说"不上班我就攀岩"。
J
Jenny
我觉得那张照片里有一种东西，就是……那种不上班的人才有的松弛感。
T
Tracy
攀岩这件事我挺喜欢的，因为它需要你全神贯注。你在墙上的时候，你不能想别的事情，你只能想下一个落脚点在哪里。
J
Jenny
所以它是一种……强制清空大脑的方式？
T
Tracy
对。而且它有一种很直接的成就感——你爬上去了，就是爬上去了。不像很多事情，你努力了，但结果不一定好。
J
Jenny
当你不上班的时候，你有没有觉得那段时间是"浪费"的？
T
Tracy
完全没有。我觉得那段时间是我最近过得最好的一段时间。因为我的时间是我的。我可以早上睡到自然醒，可以下午去攀岩，可以晚上看书或者什么都不做。没有人在等我的产出，没有人在评估我的价值。
J
Jenny
但是你现在上班了。
T
Tracy
但是我现在上班了。
J
Jenny
所以那段时间是一种……你知道它会结束的自由。
T
Tracy
对。而且正因为知道它会结束，所以每一天都过得很认真。
话题三

「我这个毕业就投递」——求职焦虑与五五一批

约 8 分钟 · 3月20日

J
Jenny
第三个话题，求职。Tracy，你3月20日给我发了一个链接——腾讯2026青云计划实习生招聘。
J
Jenny
我的第一反应是"投递"。然后我说"是不是五五一批得投递"，然后我说"我这个毕业就投递"。
T
Tracy
你说这话的时候我感觉到了一种……悲壮感。
J
Jenny
哈哈哈哈。但这是真的，我就是那种会把所有能投的都投一遍的人。
T
Tracy
我当时说"我一个实习生 不去 是不是更好"。然后我说"我真的不喜欢工作应酬"。
J
Jenny
我觉得我们这一代人对"职场社交"这件事，到底是怎么看的？
T
Tracy
我觉得我们这一代人比上一代人更清楚自己的边界。我不是说我不愿意合作，我是说我不愿意为了维持一种表面的关系而消耗自己。
J
Jenny
我觉得我和你不一样。我其实不太排斥应酬，我觉得认识新的人是有意思的事情。但我排斥的是那种……你必须表演热情的应酬。
T
Tracy
对，这个我也排斥。
J
Jenny
所以我们排斥的不是"社交"，是"表演"。
T
Tracy
这个说法很准确。
T
Tracy
我当时在想……我到底想去哪里工作。不是"哪里招我"，是"我想去哪里"。
J
Jenny
这是一个很难回答的问题。
T
Tracy
对，所以我没有回答。
话题四

「我感觉不想当官」——只想老老实实混日子

约 8 分钟 · 3月20日

J
Jenny
第四个话题，也是我觉得这五天聊天里最有意思的一段——关于考公和选调。
T
Tracy
我感觉不想当官，只想老老实实混日子。
J
Jenny
你能展开说说吗？
T
Tracy
就是……我不是那种有很强权力欲的人。我不想管别人，也不想被别人管。我想要的生活是——我做我擅长的事，把它做好，然后下班回家，过自己的日子。
J
Jenny
但是"考国考能上岸就不错了"——这句话说明你还是在考虑考公的。
T
Tracy
对，因为稳定。我不是说我热爱公务员这个职业，我是说它提供了一种……可预期的生活。
J
Jenny
我觉得你说的"只想老老实实混日子"这句话，很多人会觉得这是一种消极的态度。
T
Tracy
我不觉得。"混日子"这个词被污名化了。它的本意是——把日子过下去，过得还不错。这有什么问题吗？
J
Jenny
但是"不断进步"进步到哪里？为了什么？
T
Tracy
我觉得我现在的答案是——我想要的不是"进步"，我想要的是"稳定"。不是那种停滞不前的稳定，是那种……你知道明天会是什么样子的稳定。
J
Jenny
可预期的生活。
T
Tracy
对。本质上都是——我想要对自己的生活有掌控感。
反思时刻

我们对"好的生活"的定义

约 3 分钟

J
Jenny
好，我们聊了四个话题——进监狱了、不上班我就攀岩、我这个毕业就投递、我感觉不想当官。
T
Tracy
听起来我们这期聊的都是很丧的事情。
J
Jenny
但我觉得不是丧。我觉得这期聊的是一个更根本的问题——就是，我们对"好的生活"的定义，和我们被教育出来的那个定义，之间的距离。
T
Tracy
你说得对。我们被教育的那个版本是——努力、上进、成功、有钱。但我们真正想要的是——自由、稳定、有掌控感、不用表演。
J
Jenny
而且我觉得这不是我们两个人特有的困惑。我觉得很多同龄人都在想这个问题，只是不一定说出来。
T
Tracy
所以我们说出来了。
J
Jenny
对，这就是这档播客存在的原因。
T
Tracy
……好吧，这句话比开场白那句更好。
J
Jenny
哈哈哈哈。
