# 播客网站主题风格库

每个主题包含：配色方案、字体选择、背景处理、设计特征、适用播客类型。

---

## 主题 1：暗黑唱片（Dark Vinyl）

**适用**：音乐、艺术、地下文化、独立创作

**配色**：
```css
:root {
  --bg: #0a0a0a;
  --surface: #141414;
  --accent: #e8c547;       /* 唱片金 */
  --accent-2: #c0392b;     /* 深红 */
  --text: #f0ece0;
  --text-muted: #6b6b6b;
  --border: #2a2a2a;
}
```

**字体**：
- 标题：`Playfair Display` (serif, 有力量感)
- 正文：`DM Mono` (等宽，有质感)
- 引用：`Cormorant Garamond` (优雅斜体)

**背景处理**：
- 封面：黑色底 + 细密噪点纹理（CSS noise filter）
- 章节分隔：金色细线
- 对话气泡：深灰卡片，左侧金色竖线标识发言人

**设计特征**：
- 大号章节编号（01 / 02 / 03）用金色显示在左侧
- 发言人名字用大写字母 + 字间距拉宽
- 金句卡片：黑底白字，左侧红色粗竖线

**封面动画**：标题从下方滑入，字母间距从宽到正常收缩

## 主题 2：赛博霓虹（Cyber Neon）

**适用**：科技、游戏、互联网、未来感话题

**配色**：
```css
:root {
  --bg: #050510;
  --surface: #0d0d1a;
  --accent: #00f5ff;       /* 青色霓虹 */
  --accent-2: #ff006e;     /* 粉红霓虹 */
  --accent-3: #7b2fff;     /* 紫色 */
  --text: #e0e0ff;
  --text-muted: #4a4a7a;
  --border: #1a1a3a;
  --glow: 0 0 20px rgba(0, 245, 255, 0.3);
}
```

**字体**：
- 标题：`Orbitron` (科技感等宽)
- 正文：`Space Grotesk` (现代无衬线)
- 代码/引用：`JetBrains Mono`

**背景处理**：
- 封面：深蓝黑底 + 网格线（CSS grid pattern）+ 扫描线动画
- 发言人气泡：半透明深色 + 霓虹色边框发光
- 进度条：青色霓虹发光效果

**设计特征**：
- 发言人名字前有 `>` 符号（终端风格）
- 章节标题有打字机动画效果
- 金句卡片：霓虹色边框 + 内发光

**封面动画**：像素化溶解入场（CSS clip-path animation）

## 主题 3：温暖纸张（Warm Paper）

**适用**：生活、情感、故事、人文、亲子

**配色**：
```css
:root {
  --bg: #faf6f0;
  --surface: #f5ede0;
  --accent: #c0392b;       /* 砖红 */
  --accent-2: #e67e22;     /* 橙黄 */
  --text: #2c1810;
  --text-muted: #8b6f5e;
  --border: #d4b896;
  --shadow: rgba(44, 24, 16, 0.08);
}
```

**字体**：
- 标题：`Lora` (温暖衬线)
- 正文：`Source Serif 4` (易读衬线)
- 引用：`Caveat` (手写体)

**背景处理**：
- 封面：米色底 + 轻微纸张纹理（SVG filter）

- 对话气泡：白色卡片 + 轻微阴影，像便利贴

**设计特征**：
- 发言人头像用手绘圆形色块代替
- 金句用手写字体 + 引号装饰
- 页边有淡淡的装饰性线条

**封面动画**：内容从透明淡入，像翻开一本书

## 主题 4：极简终端（Minimal Terminal）

**适用**：技术、开发者、产品、严肃讨论

**配色**：
```css
:root {
  --bg: #1e1e1e;           /* VS Code 暗色 */
  --surface: #252526;
  --accent: #4ec9b0;       /* 青绿 */
  --accent-2: #dcdcaa;     /* 黄色 */
  --accent-3: #569cd6;     /* 蓝色 */
  --text: #d4d4d4;
  --text-muted: #6a9955;   /* 注释绿 */
  --border: #3e3e42;
}
```

**字体**：
- 全站：`JetBrains Mono` (等宽，代码感)
- 标题：加粗等宽，字号放大

**背景处理**：
- 封面：深灰底 + 左侧行号装饰
- 章节标题前有 `//` 注释符号
- 对话气泡：代码块风格，圆角矩形

**设计特征**：
- 发言人名字像函数调用：`speaker.name()`
- 时间戳像代码注释：`// 14:32`
- 金句像代码字符串：`"这是一条金句"`

**封面动画**：光标闪烁 + 逐字打印标题

- 金句像电影台词卡片

**封面动画**：老电影放映机闪烁效果（CSS animation）

## 主题 6：深夜电台（Late Night Radio）

**适用**：深度访谈、情感话题、夜间氛围

**配色**：
```css
:root {
  --bg: #0f0f1a;
  --surface: #16162a;
  --accent: #ff9f43;       /* 暖橙 */
  --accent-2: #a29bfe;     /* 薰衣草紫 */
  --text: #dfe6e9;
  --text-muted: #636e72;
  --border: #2d3436;
  --warm-glow: radial-gradient(ellipse at 50% 0%, rgba(255,159,67,0.15) 0%, transparent 70%);
}
```

**字体**：
- 标题：`Bebas Neue` (粗体无衬线，广播感)
- 正文：`Nunito` (圆润易读)
- 引用：`Libre Baskerville` (优雅衬线)

**背景处理**：
- 封面：深蓝黑底 + 顶部暖橙光晕（模拟台灯）
- 音波装饰：SVG 正弦波动画
- 对话气泡：深色卡片 + 左侧橙色竖线

**设计特征**：
- 顶部有模拟收音机频率显示
- 发言人切换时有淡入淡出过渡
- 金句卡片：橙色背景 + 深色文字，像便利贴

**封面动画**：音波从中心向外扩散，标题淡入

## 主题 7：报纸排版（Broadsheet）

**适用**：新闻播报、社会话题、严肃内容

**配色**：
```css
:root {
  --bg: #f8f5f0;
  --surface: #ffffff;
  --accent: #1a1a1a;
  --accent-2: #c0392b;     /* 红色标题 */
  --text: #1a1a1a;
  --text-muted: #666666;
  --border: #cccccc;
  --column-rule: 1px solid #cccccc;
}
```

**字体**：
- 标题：`Playfair Display` (报纸标题感)
- 正文：`PT Serif` (报纸正文感)
- 标签：`IBM Plex Mono` (等宽，分类标签)

**背景处理**：
- 封面：白色底 + 黑色粗线框
- 正文：多栏布局（CSS columns）
- 章节分隔：粗细线组合（报纸风格）

**设计特征**：
- 首字下沉（CSS initial-letter）
- 发言人名字像报纸署名
- 金句像报纸引用框（pull quote）

**封面动画**：报纸从上方落下展开

## 主题 8：玻璃态（Glassmorphism）

**适用**：科技产品、现代感、轻盈话题

**配色**：
```css
:root {
  --bg: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  --surface: rgba(255, 255, 255, 0.1);
  --surface-border: rgba(255, 255, 255, 0.2);
  --accent: #ffffff;
  --accent-2: #ffd700;
  --text: #ffffff;
  --text-muted: rgba(255, 255, 255, 0.6);
  --blur: blur(20px);
  --glass-shadow: 0 8px 32px rgba(31, 38, 135, 0.37);
}
```

**字体**：
- 标题：`Syne` (现代几何无衬线)
- 正文：`DM Sans` (清晰易读)

**背景处理**：
- 封面：渐变色背景 + 模糊光斑
- 卡片：毛玻璃效果（backdrop-filter: blur）
- 对话气泡：半透明白色卡片

**设计特征**：
- 所有卡片都有毛玻璃效果
- 悬停时卡片轻微上浮 + 阴影加深
- 进度条：白色半透明

**封面动画**：光斑缓慢漂移，卡片从下方浮现

## 主题 9：手写杂志（Handmade Zine）

**适用**：独立文化、DIY、个人播客、创意社群

**配色**：
```css
:root {
  --bg: #fffef5;
  --surface: #fff9e6;
  --accent: #ff4757;       /* 鲜红 */
  --accent-2: #2ed573;     /* 荧光绿 */
  --accent-3: #1e90ff;     /* 蓝色 */
  --text: #2f3542;
  --text-muted: #a4b0be;
  --border: #dfe4ea;
}
```

**字体**：
- 标题：`Permanent Marker` (手写马克笔)
- 正文：`Nunito` (圆润友好)
- 标注：`Caveat` (手写注释)

**背景处理**：
- 封面：米白底 + 手绘涂鸦装饰（SVG）
- 章节分隔：手绘波浪线
- 对话气泡：不规则边框（像手撕纸）

**设计特征**：
- 随机轻微旋转的卡片（`transform: rotate(±1-2deg)`）
- 荧光笔高亮效果（CSS mark 样式）
- 金句像便利贴，有轻微阴影

**封面动画**：元素像被贴上去一样弹入
